MESSAGE

はじめまして。Web制作が好きで香港出身のLEUNG SIN LINGと申します。

制作を通して、達成感を求めている人間です。 様々なWebサイトが作れるようになったら、その達成感はきっとされに強くなろう。

そう思って、自分の技術を磨くために日本に来ました。

学習の成果をこのサイトで是非ご覧ください。

SKILLS

  • デザイン

    Photoshop、Illustratorなどのソフトを使用して、デザインを作成いたします。WEBサイトのインターフェースから印刷物のグラフィックまで制作の経験を持っています。

  • コーディング

    HTML / CSS / JavaScriptを使用してWebサイトを構築いたします。様々なサイトのコーディングをする経験があります。レスポンシブデザインも対応できます。

    WEBサイトの内容により動画の制作が必要な場合があります。動画の撮影（スマートフォンやビデオカメラで）と編集などのビジュアルデザインを行っていたします。
